# Enterprise Active Directory & Security Operations Home Lab

## Overview

This project simulates a real-world enterprise IT environment focused on:

- Active Directory administration
- Identity and Access Management (IAM)
- Group Policy management
- Endpoint deployment
- SIEM monitoring
- Enterprise networking
- Windows and Linux infrastructure

The lab environment was built using VirtualBox, Windows Server, Ubuntu Server, Docker, Wazuh, and Windows 11 enterprise endpoints.

---

## Infrastructure Components

### RW-DC01
Windows Server Domain Controller providing:

- Active Directory Domain Services
- DNS services
- Organizational Units (OUs)
- User and group management
- Group Policy Objects (GPOs)

### Ubuntu Server
Linux infrastructure server running:

- Docker
- Portainer
- Wazuh SIEM
- Nextcloud
- Pi-hole

### HR-PC01
Windows 11 enterprise workstation used to simulate:

- Corporate employee endpoint
- Domain-joined workstation
- Centralized authentication
- Enterprise policy management
- SIEM-monitored endpoint
